Welcome

Welcome to Lingo Logs, I thank you for using it. This software allows you to learn a language any way you want without subscriptions and adverts. To get familiar with Lingo Logs you can start with the next chapter and start your tour. If you just only want a quick tour than continue on to the next sections. Have fun learning!

Languages Bar (Quickly)

The Languages Bar is the second bar from the top, the one with all the languages. It includes all the languages currently in Lingo Logs. By clicking on the language you want to learn you will be taken to that language's homepage. From there you can select all the different chapters and learn the language.
