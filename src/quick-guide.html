    Review

    This section will just be like a quick refresher to how to use Lingo Logs. If you want a more detailed guide continue on with the next sections.

    Languages Navbar

    The little bar that you see at the very top it called the Languages Navbar. This is how you will navigate to each language to see their documentation. You will be using Languages Navbar a lot when learning a language. If you hover over it and begin to scroll it will reveal more languages.

    Chapter Sidebar

    The vertical bar on the very left is called the Chapter Sidebar. This is how we split up content into bite sized pieces and it is always visible. By clicking on a link it will take it to its respected page and the same links will be there. Well as long as you are in the same area. The chapters names will give you a hint at what its about so you can decide if you want to learn that topic. Also you can select which ever one you want as they don't require you going to other chapters first.

    Page Content

    There is really no explaining needed but its where all the pages content is and how you will learn a language. Now many pages will have many things and if you need more information on the things possible on the page go to the "Page Content" Section.
